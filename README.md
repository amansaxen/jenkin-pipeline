# Jenkins Pipeline - Hello World App

Simple Hello World web application with Jenkins CI/CD pipeline.

## Features

- Hello World web application
- Jenkins pipeline for automated deployment
- Docker containerization

## Setup

1. Clone the repository
2. Install dependencies: `npm install`
3. Run locally: `npm start`

## Jenkins Pipeline

The Jenkinsfile contains the CI/CD pipeline configuration that:
- Checks out code from GitHub
- Installs dependencies
- Runs tests
- Builds Docker image
- Deploys the application

## Access

After deployment, access the application at: `http://localhost:3000`
